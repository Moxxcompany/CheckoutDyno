# DynoPay - Payment Solutions App

## Project Overview
A Next.js TypeScript payment application with MUI components, Redux state management, and i18n support.

## Tech Stack
- **Frontend**: Next.js 14, React 18, TypeScript
- **UI Library**: Material UI (MUI) v5
- **State Management**: Redux Toolkit + Redux Saga
- **Internationalization**: next-i18next
- **Styling**: CSS (globals.css with custom fonts)

- `/app/Components/` - Reusable UI components
- `/app/Containers/` - Page containers (Payment)
- `/app/Redux/` - State management
- `/app/contexts/` - React contexts (Theme)
- `/app/styles/` - Global styles
